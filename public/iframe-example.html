챗봇 iframe 통합 예시

이 페이지는 챗봇을 iframe으로 다른 웹사이트에 통합하는 방법을 보여줍니다.

📋 사용 방법

아래 코드를 복사하여 원하는 웹페이지에 붙여넣으세요:

<iframe src="http://localhost:5173/embed/chat/service1" width="100%" height="600px" frameborder="0" title="챗봇" ></iframe>

💡 특징: 임베드 모드에서는 뒤로가기 버튼이 표시되지 않아 사용자가 다른 서비스로 이동할 수 없습니다.

🔧 설정 옵션

  • service1: 서비스 ID (실제 서비스 ID로 변경)
  • width: iframe 너비 (100%, 400px 등)
  • height: iframe 높이 (600px, 100vh 등)
  • frameborder: 테두리 (0 또는 1)

🔒 보안 특징

  • 뒤로가기 버튼 없음: 사용자가 다른 서비스로 이동할 수 없음
  • 최소화/최대화: 임베드 환경에서만 사용 가능
  • 독립적 실행: 외부 사이트에서 완전히 격리된 챗봇

📱 반응형 설정

모바일에서도 잘 보이도록 CSS를 추가하세요:

.chat-iframe { width: 100%; height: 70vh; min-height: 400px; border: none; border-radius: 8px; } @media (max-width: 768px) { .chat-iframe { height: 60vh; min-height: 300px; } }
